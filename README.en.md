# fileup

#### Description
fileup项目来自于陆二零网络科技公司开源产品，fileup是在局域网分享的软件。在一个局域网下，文件拖拽入fileup内生成下载链接和二维码，其他局域网内的设备即可快速下载，打破带宽限制，实现真正的文件传输！
